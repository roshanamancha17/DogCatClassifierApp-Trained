# 🐶🐱 Dog vs Cat Classifier-Trained

A simple Flask web app to classify uploaded images as either a **Dog** or a **Cat** using a trained CNN model (`dogs_vs_cats_classifier.h5`).

## 🔧 Tech Stack
- Python
- TensorFlow / Keras
- Flask
- HTML + CSS + JS

## 🚀 How to Run

```bash
pip install -r requirements.txt
python app.py
